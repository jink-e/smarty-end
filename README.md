# Smarty-end

> Smarty是一款基于JS技术栈的低代码平台！

- 前后端分离架构 
- Koa2，Typescript，Vite2.0，Vue3.0 , Element3 ,
- 通过数据模型定制器实现低代码开发
- 支持多用户多角色权限分配
- 支持插件扩展例如: blog插件电商插件、小程序插件
## 目录结构

```bash
.
├── backend    后端NodeJS
├── frontend   前端Admin界面
```
